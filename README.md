# CineBuzz Premium Prototype

Prototype statique en HTML, CSS et JavaScript pour presenter une piste de redesign du site CineBuzz.

## Contenu

- `index.html` : page vitrine et entree du prototype.
- `films.html` / `film.html` : catalogue et fiche film dynamique.
- `reservation.html` : choix film, seance, billets, sieges et snacks.
- `paiement.html` : paiement simule MaxiCash.
- `confirmation.html` : ticket numerique et confirmation.
- `connexion.html` / `inscription.html` / `compte.html` : parcours compte client simule.
- `styles.css` : direction visuelle inspiree du projet Stitch "Cinebuzz Premium UI Redesign".
- `app.js` : donnees demo, navigation dynamique, reservation et compte localStorage.
- Theme clair/sombre : bouton dans le menu, choix conserve dans `localStorage`.
- `assets/` : logo CineBuzz, image hero et visuels publics utilises pour la demonstration.

## Sources de contexte

- Site public : https://cinebuzzcd.com/
- Application Android officielle : https://play.google.com/store/apps/details?id=com.defmaks.cinebuzz
- Donnees observees sur le site actuel : salles UTEX et Premier Shopping Mall, versions VF/VOSTFR, paiement MaxiCash, tarifs adultes/enfants/couples.
- Films actualises pour la demo : Michael, Le Diable s'habille en Prada 2, Mortal Kombat II et The Super Mario Galaxy Movie.

## Utilisation

Ouvrir `index.html` directement dans un navigateur ou publier le dossier sur GitHub Pages. Le prototype est sans backend : le compte, le panier et le billet sont simules dans le navigateur.
